🚗 Real-time Car Counting

Computer Vision Project

Interactive vehicle counting system with real-time visualization on interactive maps

🌟Project Overview

This web application provides real-time vehicle counting on a road, displayed in 4 interactive frames. The application visualizes traffic conditions using a map, vehicle counts, pie charts, and time series data. The system updates every few seconds to provide accurate traffic insights.

Key Features

  • Map Visualization: Displays road with color-coded traffic conditions that update every 2 seconds
  • Vehicle Detection: Real-time counting and classification of vehicles
  • Vehicle Classification: Categorizes vehicles into buses, cars, motorcycles, and trucks
  • Dynamic Charts: Time series and pie charts that update automatically with new data
  • Traffic Prediction: Uses vehicle data to predict road congestion
